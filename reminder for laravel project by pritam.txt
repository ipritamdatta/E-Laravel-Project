Full documentation is here: https://github.com/Crinsane/LaravelShoppingcart (if you have laravel 5.6.*)
Or,
try this, 
https://github.com/darryldecode/laravelshoppingcart (if you have laravel 5.8.*)



To create cart you have to run this command at F:\xampp\htdocs\elaravel here: 
composer require gloudemans/shoppingcart 

Example:
if you have laravel 5.6.* than run the following command: 
 F:\xampp\htdocs\elaravel>composer require gloudemans/shoppingcart
Or, if you have laravel 5.8.* than run the following command: 
 F:\xampp\htdocs\elaravel>composer require "darryldecode/cart:~4.0" 

It will be available in composer.json file.
